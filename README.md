# SO_2019_EP1

## Java e Git

### Compilar

```sh
javac *.java
```

### Rodar

```sh
java Main
```

### Desenvolver nova feature

```sh
git pull
git checkout DEV
git branch Categoria_feature
git checkout Categoria_feature
```

### Incluir nova feature

```sh
git checkout DEV
git merge Categoria_feature
git push
```
